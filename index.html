# Deploying Pet Crew's legal docs (GitHub Pages) The app links to a hosted **Terms of Service** and **Privacy Policy**. The source markdown lives in this repo at `legal/terms-of-service.md` and `legal/privacy-policy.md`; this doc explains how to publish them and where the app expects them to live. ## Target URLs (what the app links to) The app (`lib/config.ts` → `LEGAL_URLS`) points at: - Terms: `https://unpaidmajdoor.github.io/petcrew-legal/terms` - Privacy: `https://unpaidmajdoor.github.io/petcrew-legal/privacy` If you deploy to different URLs, update `LEGAL_URLS` in `lib/config.ts` to match. ## Setup — dedicated `petcrew-legal` GitHub Pages repo 1. Create a new public repo named **`petcrew-legal`** under the `unpaidmajdoor` account. 2. Copy `legal/terms-of-service.md` and `legal/privacy-policy.md` into it. Convert each to HTML (any markdown-to-HTML tool works), or keep the markdown and let a GitHub Pages theme render it. 3. Add an `index.html` at the repo root that links to both pages (sample below). 4. To get the clean `/terms` and `/privacy` paths, create either: - `terms.html` and `privacy.html` at the repo root, **or** - folders `terms/index.html` and `privacy/index.html`. 5. Enable **Settings → Pages → Deploy from branch → `main` / root**. 6. Push. The pages go live at `https://unpaidmajdoor.github.io/petcrew-legal/terms` and `.../privacy` within a minute or two. ## Sample `index.html` ```html Pet Crew — Legal

Pet Crew — Legal

Pet Crew is operated by Sanket Maharana, based in Bengaluru, India.

  • Terms of Service
  • Privacy Policy

Contact: maharana.mds92@gmail.com

``` Convert each markdown file to HTML and save as `terms/index.html` and `privacy/index.html` (or `terms.html` / `privacy.html`). ## Play Console prep - The Google Play Console listing **requires a Privacy Policy URL**. - Use: `https://unpaidmajdoor.github.io/petcrew-legal/privacy` - Enter it under **App content → Privacy Policy** in the Play Console. - Keep the URL live and current — Google re-checks it, and a dead link can block or remove the listing.
